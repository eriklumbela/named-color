named color

named color is an art project by Erik Lumbela.
Colors featured on this website correspond to the 148 named colors as defined in the CSS specification

You can download these colors in svg and html formats below.

This website represent 'named color' in CSS, as the web is most suitable for showing CSS. The CSS specification provided the name for this series, but conceptually it is not limited to CSS. Among others there is also a 'named color' series of Copic's, and Pantone, with color names from the Copic Color System and Pantone Color Systems respectfully.
